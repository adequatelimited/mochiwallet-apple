# Resources Directory

This directory contains the built web extension assets.

**Do not manually edit files in this directory.**

Files are automatically generated by the build script (`build.sh`) from the upstream `mochiwallet` extension submodule.

## Contents (after build)

- `index.html` - Main entry point
- `assets/` - JavaScript, CSS, and other assets
- `polyfills.js` - Chrome API compatibility layer
- `ios-ui.css` - iOS-specific styling
- `hide-*.js` - UI modification patches

## Rebuilding

To regenerate these files:

```bash
cd ..  # Go to mochiwallet-apple root
./build.sh
```
